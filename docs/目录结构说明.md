# 项目目录结构说明

## 根目录
- `README.md`: 项目主说明文档
- `build.gradle`: 项目构建配置
- `gradlew/gradlew.bat`: Gradle 启动脚本
- `src/`: 源代码目录

## docs/ 文档目录
- `README.md`: 文档主说明
- `api/`: API 接口文档
- `architecture/`: 架构设计文档
- `config/`: 配置文件目录
  - `generalized_learning/`: 通用学习配置
- `deployment/`: 部署相关文档和配置
  - `DOCKER-DEPLOY.md`: Docker 部署说明
  - `config/`: 部署配置文件
    - `Dockerfile`: Docker 构建文件
    - `docker-compose.yml`: Docker Compose 配置
    - `nginx.conf`: Nginx 配置
    - `start-docker.sh`: Docker 启动脚本
    - `start.sh`: 通用启动脚本
- `design/`: 设计文档
  - `GENERALIZED_SQL_LEARNING_DESIGN.md`: 通用 SQL 学习设计
  - `SQL_PATTERN_LEARNING_DESIGN.md`: SQL 模式学习设计
  - `Text2SQL-MCP设计方案.md`: Text2SQL MCP 设计方案
- `development/`: 开发相关文档
- `guides/`: 使用指南和说明文档
  - `依赖修复说明.md`: 依赖修复说明
  - `快速启动指南.md`: 快速启动指南
  - `恢复指南.md`: 恢复指南
  - `项目说明.md`: 项目说明
- `progress-reports/`: 进度报告
- `scripts/`: 脚本文件
  - `fix-ide.sh`: IDE 修复脚本
